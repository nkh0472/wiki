Zotero PDF Translate 是由@windingwind 开发的，用于文献管理软件 Zotero 的翻译插件。它完全开源、免费，主要功能有：

- 划词翻译（自动/手动）
- 标题、摘要翻译与源文本/翻译结果显示切换
- 批注翻译（自动/手动）
- 支持 15+种翻译服务与 4+种字典服务
- 自定义翻译侧栏 UI，支持同时使用多种翻译引擎

插件深度集成了 Zotero 文献阅读功能。

本文档主要介绍了翻译插件的基本功能，使用方法。
